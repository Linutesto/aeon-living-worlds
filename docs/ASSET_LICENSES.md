# AEON — Asset Licenses

All texture assets bundled with AEON are **CC0 1.0 (Public Domain)** sourced from
**Poly Haven** (https://polyhaven.com). CC0 places no restrictions on use or
redistribution and **requires no attribution**; the credits below are provided as a
courtesy and for provenance.

Files live in two delivery tiers:

- `web/assets/textures/<file>` — 512×512 JPG for mobile delivery, resized from the
  1K diffuse maps on the Poly Haven CDN.
- `web/assets/textures/2k/<file>` — 2048×2048 JPG for desktop and ultra-4090, downloaded
  directly from the matching 2K diffuse maps on the Poly Haven CDN.

The table below documents both local tiers by filename convention: each listed local

## How these were obtained

Downloaded from the Poly Haven CDN:

- Mobile tier: `https://dl.polyhaven.org/file/ph-assets/Textures/jpg/1k/<asset>/<asset>_diff_1k.jpg`,
  then resized to 512px JPG (quality 82).
- Desktop/ultra tier: `https://dl.polyhaven.org/file/ph-assets/Textures/jpg/2k/<asset>/<asset>_diff_2k.jpg`,
  stored as 2048px JPG.
