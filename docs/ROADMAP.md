# Roadmap

Staged milestones for AEON: Living Worlds. This is a public-prototype roadmap — order and
scope will shift as the project evolves. The longer design vision lives in the top-level
[../ROADMAP.md](../ROADMAP.md).

Legend: ✅ done · 🔨 in progress · 📋 planned

## M0 — Public prototype ✅
The world runs, renders, restarts, and saves; repo is clean and documented for public
release. Deterministic worldgen, plural civilizations, emergent society, optional LLM
spirit, test suite.

## M1 — Visual polish & placement 🔨
Collision-free city layout (done, hardening ongoing), richer silhouettes and district
identity, fewer overlap edge-cases in dense terrain, better distant impostors, and texture
work. → [KNOWN_LIMITATIONS.md](KNOWN_LIMITATIONS.md)

## M2 — Civilization lifecycle 🔨
Deepen collapse / merge / split / vassalize / golden-age dynamics; make diplomacy and
succession legible in the UI (atlas, map overlays, history).

## M3 — Citizen individuality 📋
Richer personalities, relationships, ambitions, and life-arcs; more meaningful focus/follow
and interviews; readable per-citizen consequences of macro events.

## M4 — Historical memory 📋
Persistent historical sites and disaster scars feeding back into settlement appeal,
diplomacy, religion, and unrest — history that shapes the future, not just a log.

## M5 — AI behavior systems 📋
Promote the prototype minds: stable, legible per-species behavior (drift-controlled via
AWR + KL trust region), smarter cohort routing, and clearer "why did they do that".

## M6 — Modding & texture packs 📋
True custom per-pack asset sets, a documented pack/mod format, and data-driven content so
the community can theme and extend worlds. → [TEXTURE_PACKS.md](TEXTURE_PACKS.md)

## M7 — Performance & mobile polish 📋
Hit 60 FPS desktop / 30 FPS mobile consistently in dense cities: GPU-driven culling,
atlasing, tighter LOD, and a refined mobile UI.

## M8 — Trailer / demo release 📋
A captured trailer, a hosted or one-click demo, and a tagged release with screenshots and
sample worlds. → [TRAILER_CAPTURE.md](TRAILER_CAPTURE.md)
